# Code book for Gettin and Cleaning Data course project

## All measurements are normalized so they have no unit.

### activity
Activity label for test activity, one of:
* WALKING
* WALKING_UPSTAIRS
* WALKING_DOWNSTAIRS
* SITTING
* STANDING
* LAYING

### subject
Numeric identifier for test person (1-30)
